# Usual data project structure

## Pre requisites:
1. python
2. pyenv
3. poetry

## Instaling libraries and virtual environment

In project folder
1. `pyenv install 3.12.5`
2. `poetry env use 3.12.5`
3. `poetry shell`
4. `poetry install`

Your virtual environment and depenencies should be installed and running.

## Running the project

1. python main.py
